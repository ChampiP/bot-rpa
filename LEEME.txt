================================================================================
MENU PRINCIPAL
================================================================================

El archivo BOT_RPA_CLARO.bat tiene TODO integrado:

[1] Ejecutar Bot (Interfaz Grafica)
    -> Abre la GUI amigable
    -> Configura y ejecuta desde ahi
    -> Recomendado para usuarios normales

[2] Ejecutar Bot (Modo Consola)
    -> Ejecucion directa en terminal
    -> Ves todo el proceso en tiempo real
    -> Para usuarios avanzados

[3] Instalar / Reparar Bot
    -> Primera instalacion
    -> Reparar si algo falla
    -> Actualizar dependencias

[4] Verificar Sistema
    -> Diagnostico completo
    -> Ver si todo funciona
    -> Detectar problemas

[5] Salir

================================================================================
PRIMERA VEZ - INSTALACION
================================================================================

1. Ejecuta BOT_RPA_CLARO.bat
2. Selecciona opcion [3] Instalar
3. Espera 2-5 minutos
4. ¡Listo! Ahora usa opcion [1] para ejecutar

================================================================================
USO DIARIO
================================================================================

1. Doble clic en BOT_RPA_CLARO.bat (Escritorio)
2. Opcion [1] - Interfaz Grafica
3. Configura credenciales (primera vez)
4. Click en "EJECUTAR BOT"
5. ¡Disfruta!

================================================================================
CONFIGURACION DE CREDENCIALES
================================================================================

OPCION A: Desde la GUI (Mas Facil)
1. Ejecuta BOT_RPA_CLARO.bat -> Opcion [1]
2. Pestana "Ajustes"
3. Ingresa usuario y contraseña
4. Click "Guardar"

OPCION B: Editar .env directamente
1. Abre el archivo .env con un editor de texto
2. Pon tus credenciales:
   CLARO_USUARIO=tu_usuario
   CLARO_CLAVE=tu_contraseña
3. Guarda

================================================================================
CONFIGURACION AVANZADA (OPCIONAL)
================================================================================

Ejecuta BOT_RPA_CLARO.bat -> Opcion [1] -> Pestana "Avanzado"

Puedes configurar:
- Modo Debug (para ver detalles)
- Proxy (para redes corporativas)
- Timeouts (para conexiones lentas/rapidas)

================================================================================
SOLUCIONES RAPIDAS
================================================================================

PROBLEMA: No funciona nada
SOLUCION: BOT_RPA_CLARO.bat -> Opcion [3] Instalar/Reparar

PROBLEMA: Python no encontrado
SOLUCION: BOT_RPA_CLARO.bat -> Opcion [3] -> Opcion [1] instalar automatico

PROBLEMA: Bot muy lento
SOLUCION: GUI -> Pestana Avanzado -> Disminuir timeouts

PROBLEMA: Timeouts frecuentes
SOLUCION: GUI -> Pestana Avanzado -> Aumentar timeouts

PROBLEMA: Necesito diagnostico
SOLUCION: BOT_RPA_CLARO.bat -> Opcion [4] Verificar
